The "All_datasets" folder contains all the important datasets that were used. 





The "Experiments" folder contains all the major important experiments done. The most important experiments that are talked about in the report are those under the "5x" folders of each model.




The script folder contains all the major scripts used, from dataset treatment, to exploratory data analysis to other less important scripts.




The actual training scripts can be found in the logs folders as every time a training was made, a copy of the training script was always saved under its respective log folder.




The Training_Script_example.py explains how a script works, all of them follow the same structure.
